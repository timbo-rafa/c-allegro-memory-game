### Compilation

##### Allegro

```bash
sudo apt-get install liballegro4.4 liballegro4-dev
```

Allegro *4* is required. You can download it [here](https://liballeg.org/old.html).

##### Make

```
make
```

### Running
```
./memory-game
```
